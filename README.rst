seu-wlan
=========

seu-wlan 是帮助自动登录校园网并防止掉线的小工具

Requirements
------------
Go 1.11

Installation
------------
$ 

Usage
-----
usage: seu-wlan -u username -p password [-i seconds]

positional arguments:
  username                一卡通号码
  password                统一认证密码

optional arguments:
  -i                      设置轮询登录间隔，以秒为单位
